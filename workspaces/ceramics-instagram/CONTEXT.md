# Task Routing

| I want to... | Go to |
|--------------|-------|
| Set up my brand identity for the first time | `setup/questionnaire.md` |
| Add new photos/videos for this week's content | `stages/01-input/CONTEXT.md` |
| Decide what to post this week | `stages/02-planning/CONTEXT.md` |
| Write captions for my posts | `stages/03-content/CONTEXT.md` |
| Create stories or reels from my content | `stages/04-repurposing/CONTEXT.md` |
| Review or update my brand identity | `brand-vault/identity.md` |
| Check my voice guidelines | `brand-vault/voice-rules.md` |
| Find hashtag sets to use | `shared/hashtag-library.md` |

## Quick Start

New week? Start here:
1. `stages/01-input/CONTEXT.md` -- collect your assets
2. Follow the pipeline through stages 02-04

First time? Run `setup` to configure your brand identity.
