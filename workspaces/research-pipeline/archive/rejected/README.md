## Learning from Rejections

Each rejection is a learning opportunity. Review patterns:

1. **Common failure modes:**

   - Depth insufficient → Expand discovery
   - Actionability lacking → Focus synthesis on recommendations

2. **Do NOT add to KB:** Research in this folder should not be integrated into the knowledge base.

3. **Re-run allowed:** If the topic is important, re-run research with methodology adjustments and log as an experiment.
