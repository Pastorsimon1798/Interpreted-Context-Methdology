### Credibility Scoring

Sources were evaluated using the credibility scoring system:

| Factor | Points |
|--------|--------|
| Trusted domain (.edu, .gov, .org) | +30 |
| Academic/research path in URL | +10 |
| HTTPS enabled | +5 |
| High-engagement (HN >100 pts) | +5 |
| Suspicious TLD (.xyz, .tk) | -20 |
| No HTTPS | -10 |

**Threshold:** Sources below 40 points were excluded from conclusions.
