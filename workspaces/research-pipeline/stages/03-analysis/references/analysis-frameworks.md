# Analysis Frameworks

Patterns for categorizing and extracting insights from sources.

---

## Categorization Dimensions

**By theme:** What topic area does this finding address?
**By type:** Is this data, opinion, methodology, or conclusion?
**By quality:** How strong is the evidence?
**By recency:** Is this current or historical context?

---

## Finding Extraction Template

For each source, extract:
- **Key claim:** What is the main assertion?
- **Evidence:** What supports this claim?
- **Method:** How was this determined?
- **Limitations:** What weaknesses exist?
- **Relevance:** Which research question does this address?

---

## Pattern Recognition

Look for:
- **Convergence:** Multiple sources agree
- **Divergence:** Sources disagree on same point
- **Gaps:** Questions without sources
- **Surprises:** Unexpected findings

---

## Evidence Linking

Every claim in analysis should link to:
1. Source ID
2. Specific quote or data point
3. Page/section reference if available

Format: `[Claim] (SourceID, p.X)`
